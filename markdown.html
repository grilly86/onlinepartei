Markdown

Was ist Markdown?

Markdown ist eine vereinfachte Auszeichnungssprache, mit dem Ziel, dass bereits die Ausgangsform leicht lesbar ist. Daher wurde eine Auszeichnungssyntax gewählt, die bereits in Plaintext-Files und E-Mails gebräuchlich waren.

Wie verwende ich Markdown?

Auf onlinepartei.eu ist Markdown an allen Stellen implementiert, wo Formatierungen zulässig sind (also zB bei Meldungen und Chat-Nachrichten, aber zB nicht bei Benutzernamen).

Absatz

Absätze werden durch Leerzeilen getrennt.
Aus: 

Normaler Text wird so dargestellt wie eingegeben.

Eine Leerzeile erzeugt einen Absatz.

Wird:

Normaler Text wird so dargestellt wie eingegeben.

Eine Leerzeile erzeugt einen Absatz.

Hervorhebungen

Durch Sternchen oder Unterstrichen werden Hervorhebungen erreicht: ein Sternchen für kursiv, zwei Sternchen für fett und drei Sternchen für fett und kurzsiv:
Aus: 

*Kursiv*, **Fett** und ***Fett kursiv***

bzw. (gleichbedeutend):

_Kursiv_, __Fett__ und ___Fett kursiv___

Werden zu:

Kursiv, Fett und Fett kursiv

Verlinkungen

Um Verlinkungen zu erstellen gibt es in Markdown mehrere Möglichkeiten:

Variante 1

<http://www.example.org/>

Erstellt einen einfachen klickbaren Link mit der Url als Linktext:

http://www.example.org/

Variante 2

[linktext](http://www.example.org)

Erstellt einen klickbaren Link mit einem definierten Linktext:
